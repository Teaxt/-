# 🐍 Terminal Snake

一个在 Windows 终端（命令提示符 / Windows Terminal）里运行的经典贪吃蛇游戏。  
无需安装任何第三方库，双文件结构，双击即可开玩。

---

## ✨ 功能特点

- 纯终端运行，利用 ANSI 转义序列实现彩色绘制和光标控制
- 双模式操作：同时支持 `WASD` 和 `方向键`
- 动态难度：每吃到一个食物，移动速度逐渐加快
- 实时显示当前得分与移动速度
- 完整的碰撞检测：撞墙或撞到自己均会结束游戏
- 一键启动批处理，无需手动输入命令

---

## 🛠 技术实现

- **语言**：Python 3  
- **依赖**：仅使用标准库（`os`, `sys`, `time`, `random`, `msvcrt`, `ctypes`），无第三方依赖  
- **平台**：Windows 10 / 11  
- **核心模块**  
  - `msvcrt` —— 实现非阻塞方向键监听  
  - `ctypes` —— 启用控制台虚拟终端 ANSI 支持  
  - 双缓冲绘制，避免屏幕闪烁

## 🚀 快速开始

1. 确保你的电脑已安装 **Python 3**  
   （若未安装，请访问 [python.org](https://www.python.org/downloads/) 下载并安装，安装时务必勾选 `Add Python to PATH`）
2. 下载或克隆本仓库到本地
3. 双击 `开始游戏.bat` 即可运行游戏  
   你也可以在项目目录下打开命令行，手动执行：
   ```bash
   python snake_game.py

   游戏规则

  ***

   目标
控制蛇吃到地图中红色的食物 O，每吃一个得 10 分，蛇身会增长一节。

动态难度
随着你的分数提高，蛇的移动速度会逐渐加快，越到后面挑战越大。

结束条件

蛇头撞到上下左右任意一面墙壁

蛇头碰到自己的身体
满足任一条件游戏即结束，屏幕会显示你的最终得分。

重新开始
游戏结束后按任意键退出，再次双击 开始游戏.bat 即可重新挑战。
